Front-End Admin Panel for Posts, Pages & Custom Post Types Custom Fields

== Description ==

Welcome to WP Front Admin plugin!

This plugin will enhance your administration experience & efficiency by adding a side panel which let you quickly Add/Update/Delete any meta data on Posts, Pages & Custom Post Types. Ever had trouble to debug or edit serialized post meta? WP Front Admin got your back. The plugin includes an awesome serialized data editor!

= Features: =

* Easy to Install & Uninstall.
* Add / Update / Delete any Post Meta directly on the Frontend
* Including a Serialized Data Editor

= Requirements: =

* PHP 5.3+

== Installation ==

= Wordpress Install =

1. Upload the plugin files to the `/wp-content/plugins/wp-front-admin` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go on any Post, and click on the top right button called "Front Admin"

== Frequently Asked Questions ==

= How to show the panel? =

Go on any single Post, Page or Post Type, a new button will appear in your admin top bar right corner. Click it. Enjoy :)
